SmartCart
SmartCart is a full-stack E-Commerce Product Management System designed to help users view, add, and manage products dynamically. Built using React.js, Node.js, Express.js, and MySQL, SmartCart provides a seamless interface for managing inventory while simulating AI-style rule-based automation such as low-stock alerts.
This project demonstrates full-stack development skills, including frontend-backend integration, database operations, RESTful API design, and dynamic UI updates.

Features
Product Listing: View all products with essential details like Name, Price, and Stock.
Add Products: Dynamically add new products directly from the frontend interface.
Low-Stock Alerts: System detects and highlights products with low inventory using rule-based automation.
Full-Stack Implementation: Complete separation of frontend and backend with RESTful APIs.
Responsive UI: Built with React.js for smooth and responsive user experience.

Technologies Used
Frontend: React.js, HTML5, CSS3, JavaScript
Backend: Node.js, Express.js
Database: MySQL
Tools & IDEs: VS Code, Postman, MySQL Workbench


Usage
Open the frontend in your browser.
View the list of products dynamically fetched from the backend.
Add new products using the Add Product form.
Low-stock items will be highlighted automatically.
All changes are persisted in the MySQL database.
